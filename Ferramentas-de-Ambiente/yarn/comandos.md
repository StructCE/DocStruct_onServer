## Comandos basicos Yarn

### Baixar dependências do projeto

```bash
yarn
```

### Iniciar o projeto

```bash
yarn start
```

### Criar um arquivo package.json

```bash
yarn init
```

### Criar o arquivo yarn.lock

```bash
yarn generate-lock-entry
```
